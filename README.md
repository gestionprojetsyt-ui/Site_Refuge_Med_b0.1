# 🐾 Site Officiel - Refuge Médéric (Les Animaux du Grand Dax)

Bienvenue sur le dépôt du site internet du **Refuge Médéric**. Cette application, développée avec **Streamlit**, permet de centraliser les adoptions, les dons et les conseils d'urgence pour les animaux du Grand Dax.

## ✨ Fonctionnalités
- **Adoptions en direct** : Consultation des fiches animaux synchronisées via Google Sheets.
- **Soutien solidaire** : Boutons de dons intégrés pour **HelloAsso** et **OuiJ'agis**.
- **Gestion des Urgences** : Guide complet pour les animaux perdus ou trouvés (Fourrière).
- **Mode Administration** : Accès sécurisé pour la gestion de la newsletter.

## 🛠️ Installation
Pour lancer le projet localement :
1. Clonez le dépôt.
2. Installez les dépendances : `pip install -r requirements.txt`
3. Lancez l'application : `streamlit run app.py`
